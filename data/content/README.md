# Learning Content

All study material is stored in this folder so you can expand content without editing UI code.

## Files
- `vocabulary.json` → single-word vocabulary in `{ "words": [...] }`
- `phrases.json` → short sentence/phrase cards in `{ "phrases": [...] }`
- `verbs.json` → verb list and conjugation forms in `{ "verbs": [...] }`
- `grammar-reference.json` → grammar sections and conjugation table content
- `stories.json` → Story Mode readings and comprehension questions in `{ "stories": [...] }`

## How to add content
1. Add a new object to the relevant array.
2. Keep field names consistent with existing entries.
3. Save the file and refresh the page.

No script changes are needed when you add entries that match the existing structure.
